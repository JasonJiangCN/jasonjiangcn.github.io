作用域

this

全局的this相当于window

作为对象方法的函数的this

指向对象本身，或调用某个函数的对象本身

对象原型链上的this

调用了原型链上的另一个对象的一个含有this的方法，这个this仍然指向这个对象本身，仍然可以访问自身的属性

构造器中的this

如果在构造器里调用了this，js会创建一个空的对象，这个对象的原型会指向构造器的那个函数。如果没写return语句，默认会把this返回。
